# DATA550-Group9

## Analysis 1: Summary Statistics
-`scripts/01_table.R` generates a summary table for death count and patient age, stratified by selected categorical variables sex or patient type.


## Analysis 2: Difference Tests
-`scripts/02_tests.R` does:
Two-sample t-test (mean age difference by sex) 
Two-sample z-test (death proportion difference by sex)
The analysis_2_tests_summary.csv contain the estimates of differences and t tests, and associated p values.
